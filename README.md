# 单词猎手 Word Hunter

一款轻量级的英语单词益智游戏，在 10×10 字母格子中找出隐藏的单词，输入中文释义即可通关。

**在线游玩**：https://zhangyx.github.io/word-hunter/

---

## 🎮 玩法说明

### 游戏目标
在 3 分钟内，找到尽可能多的隐藏单词。

### 操作步骤
1. **观察** — 10×10 字母矩阵中横向或纵向隐藏着英语单词
2. **发现** — 用肉眼找出认识的单词（比如 `apple`、`book`）
3. **输入** — 在底部输入框填写该单词的中文含义（如"苹果"）
4. **验证** — 点击 ↩ 或按回车提交，正确则格子变色并播放弹跳动画

### 计分规则
- 每找到一个单词，计 1 分
- 找不到时可以点击 ⏸ 暂停计时器思考
- 时间结束或全部找完显示结算画面

### 辅助功能
- **打草稿** — 鼠标/手指拖动格子可标记为珊瑚色，方便临时标注
- **闲置提示** — 超过 20 秒无操作，会随机让一个未找到的单词字母闪烁金色提示
- **同义词匹配** — 输入"书"能匹配"书本"，反之亦然

---

## 📦 词库格式

支持自定义 CSV 格式，每行一个单词：

```
单词,释义1|释义2|释义3
```

**示例**：
```
book,书|书本|书籍
apple,苹果|苹果树
```

支持 CET-4 标准格式（含词性前缀）：
```
ability,能力
abandon,v. 放弃，抛弃
```

- Google Fonts：Righteous / Space Mono / Noto Sans SC

## 📱 移动端使用

1. 打开游戏链接
2. 点击浏览器「分享」→ 「添加到主屏幕」
3. 即可像 App 一样离线使用
