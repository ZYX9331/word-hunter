# 🔍 单词猎手 / Word Hunter

> 一个让你背 CET-4 单词时不那么痛苦的网页版单词搜索游戏。
>
> **[在线玩 →](https://zyx9331.github.io/word-hunter/)**

---

## 这玩意是干嘛的

背单词最无聊的部分就是对着词表硬啃。这个游戏换了个玩法——你需要在字母矩阵里把单词找出来，然后**写出它的中文释义**才能得分。

这个过程叫"主动提取"（Active Recall），比单纯翻书看单词效率高得多。核心思路很简单：如果你真的记住了这个词，你应该能写出来它的意思。

整个项目就是一个单 HTML 文件，零依赖零构建。打开浏览器就能玩，手机电脑都行，放 U 盘里带走也没问题。

---

## 怎么玩

> 10×10 的字母矩阵里藏着最多 16 个英语单词，3 分钟内把它找出来。

**三步走：**

1. 扫描字母格，找到一个单词（比如发现了 `heart`）
2. 下面输入框里写它的中文含义（比如 `心脏` 或 `心灵`）
3. 回车提交。答对了格子变绿 ✓，计 1 分

**一些小细节：**

| 操作 | 效果 |
|---|---|
| ⏸ / ▶ 暂停 | 停掉倒计时 |
| 在格子上拖动 | 格子变成珊瑚红，用来临时标记你正在看的区域 |
| 难度滑块 | 控制提示频率——简单 5 秒、中等 10 秒、困难 20 秒 |
| 📂 上传词库 | 导入你自己的 CSV 词库 |

如果一直没操作，游戏会**闪烁金色**提示某个未找到单词的字母位置，频率取决于你选的难度。

答案检查带了点容错：`书`、`书本`、`书籍` 都算对。基本是包含关系匹配，不要求一字不差。

## 词库

内置 29 个常用词作为保底，联网时会自动加载附带的高频词库（**1640 个 CET-4 核心词**，从《2026年6月英语四级1500核心词》整理提取）。

如果你想用自己的词库，传 CSV 就行，格式超简单：

```csv
单词,释义1|释义2|释义3
book,书|书本|书籍
apple,苹果|苹果树
```

也兼容从真题资料里直接扒下来的标准格式，程序会自动去掉 `v.` `n.` `adj.` 这些标记：

```csv
abandon,v. 放弃，抛弃
ability,n. 才能，能力
```

## 技术实现

| 方面 | 怎么做的 |
|---|---|
| 构造 | 一个 HTML 文件，共 800+ 行，无框架无构建 |
| 单词放置 | 回溯算法，长单词优先，最多试 400 次 |
| 界面 | CSS Grid + 自定义属性 + 深蓝渐变背景 + 圆角发光棋盘 |
| 字体 | Righteous / Space Mono / Noto Sans SC |
| 移动端 | `visualViewport` API，键盘弹出时输入框自动贴在键盘上方，棋盘不滚动 |
| 离线 | 可添加到手机桌面全屏打开（无 service worker，就是纯粹的 HTML） |

**本地运行**（CSV 加载需要 HTTP，浏览器限制 `file://` 下的 `fetch`）：

```bash
npx serve .
# 或者
python -m http.server 8000
```

然后打开 `http://localhost:8000`。
