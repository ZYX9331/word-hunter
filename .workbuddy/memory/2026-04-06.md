# 2026-04-06

## word-hunter-v4.html 修改

对单词猎手游戏做了以下改动：
1. **字母改小写**：格子内字母存储和显示均改为小写（去掉 `text-transform: uppercase`，`fillRandom` 和 `placeWord` 改用小写字母）
2. **移除删除线**：删除 `.cell.found::after` 伪元素，找到单词只变色（mint 绿）不加删除线
3. **适配 CET4 CSV**：新增 `parseCET4Meanings()` 函数，能解析 `v. 放弃，抛弃` 这种带词性前缀的格式，提取纯中文词义；自动加载文件名改为 `cet4-high-frequency-words.csv`
4. **移动端优化**：`viewport-fit=cover`、`env(safe-area-inset-*)` 适配刘海屏、输入框 `font-size: max(16px,1em)` 防 iOS 缩放、按钮触摸区域加大、`autocapitalize="none"` 防止手机自动大写
5. **GitHub Pages 友好**：纯静态单文件，无额外依赖，CSV 加载失败静默降级到默认词库
